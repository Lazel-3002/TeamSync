# Bakım Araçları

Bu klasördeki dosyalar uygulamanın normal çalışması sırasında yüklenmez.

- `data/`: Pokémon veri dosyalarını üretmek veya güncellemek için kullanılan araçlar.
- `dev/`: Yalnızca paketlenmemiş geliştirme sürümünde kullanılan tanılama aracı.
- `patches/`: Daha önce kaynak dosyalara tek seferlik dönüşümler uygulamak için kullanılan eski betikler.

Betikler proje kökü çalışma dizini olacak şekilde çalıştırılmalıdır.
